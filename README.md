# 👋 Welcome to the SolDevMentorship GitHub Organization

Each student in the SolDevMentorship program will have their **own personal repository** under this organization. This repo should be used to track your projects, progress, and submissions throughout the program.

---

## 📌 Instructions: Set Up Your Personal Repository

Follow the steps below to create your personal repository:

### 1. Manually Create Your Repo

🔗 Go to: [https://github.com/organizations/SolDevMentorship/repositories](https://github.com/organizations/SolDevMentorship/repositories)

1. Click the green **"New"** button to create a new repository.
2. **Repository name**: Use **your GitHub username** exactly as it appears.  
   Example:  
   If your GitHub is `johndoe`, name your repo `johndoe`.
3. **Description**: Optional – feel free to write something like "SolDev Mentorship Projects"
4. Check **"Add a README file"**
5. Make sure the repository is being created **under the `SolDevMentorship` organization**, not your personal GitHub.

✅ You now have a repo like:  
`https://github.com/SolDevMentorship/your-github-username`

---

## 🛠️ Start Working Locally

```bash
# Clone your personal repo
git clone https://github.com/SolDevMentorship/your-github-username.git

cd your-github-username

# Add your files
touch hello-solana.md

# Commit and push
git add .
git commit -m "initial commit"
git push
```

---

## 🧱 Recommended Folder Structure

Keep your work organized! For example:

```
/your-github-username
  ├── week1/
  ├── week2/
  ├── final-project/
  └── README.md
```

## 💡 Tips
